
# Movie App
A movie app made in Flutter with clean architecture using BLoC, Hive and themoviedb API.

## Features
- Search feature
- Watchlist feature
- Now playing movies
- Popular movies
- Top rated movies
- Movie details
- Movie cast
- Movie reviews
- Similar movies
- On air tv shows
- Popular tv shows
- Top rated tv shows
- TV show details
- Similar tv shows
- TV show season details




## Installation
### 1. Clone the repo
### 2. Run pub get
```bash
flutter pub get
```
### 3. Add your API key into api_constants file
```bash
apiKey = 'YOUR_API_KEY';
```
### 4. Run the app
```bash 
flutter run
```
